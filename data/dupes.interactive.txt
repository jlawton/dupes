# dupes interactive
# -----------------
# 
# Mark files for deletion with an 'x'
# Available commands:
#  x          Mark a file for deletion and move to the next file
#  space      Unmark a file and move to the next file
#  X          Toggle a mark and move to the next file
#  down       Move to the next file
#  up         Move to the previous file
#  Q          Save and quit
#  tab        Move to the next file group
#  shift-tab  Move to the previous file group
